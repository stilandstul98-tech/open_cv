OpenCV Detection System

Система виявлення об'єктів що наближаються використовуючи OpenCV.

Файли:
-flow.py — базове виявлення руху через absdiff
-bird_video.py — відстеження птахів через оптичний потік  
-stereo_depth.py — вимірювання відстані через стерео камеру (фото)
-kitti_stereo.py — стерео зір на реальному відео (KITTI датасет)
-drone_finder.py — комбінація оптичного потоку і стерео зору
